# Быстрый старт

## Установка и запуск

```bash
# 1. Установите зависимости
npm install

# 2. Запустите сервер
npm start
```

Сервер запустится на `http://localhost:3100`

Откройте в браузере: `http://localhost:3100/implementation_schedule.html`

## Что делает сервер

- Отдает статические файлы (HTML, JS, CSS, изображения)
- Сохраняет состояние диаграммы Ганта через API `/api/gantt-state`
- Сохраняет информацию о компании (название, логотип) через API `/api/company-info`
