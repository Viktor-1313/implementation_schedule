# Инструкция по проверке темной темы таблицы

## Как проверить

### 1. Запустите сервер
```bash
cd 1
node server.js
```

### 2. Откройте страницу в браузере
```
http://localhost:3001/implementation_schedule.html
```

### 3. Переключите темную тему
Нажмите на кнопку переключения темы в правом верхнем углу (круглая синяя кнопка с Луной и Солнцем).

### 4. Переключитесь на вид таблицы
Нажмите на кнопку "Таблица" над диаграммой Ганта.

## Что проверить

### ✅ Контейнер таблицы
- [ ] Контейнер вокруг таблицы должен быть темным (`#1e1e1e`)
- [ ] Заголовок "Детальная таблица задач" должен быть голубым (`#42a5f5`)
- [ ] При активном виде таблицы контейнер должен иметь голубую обводку

### ✅ Общий вид таблицы
- [ ] Фон таблицы должен быть темным (`#1e1e1e`)
- [ ] Заголовки таблицы должны быть голубыми (`#42a5f5`) с белым текстом
- [ ] Ячейки таблицы должны иметь темный фон с светлым текстом (`#e0e0e0`)

### ✅ Интерактивные элементы
- [ ] При наведении на строку фон меняется на еще более темный
- [ ] Четные строки имеют чуть более темный фон
- [ ] При выделении строки (клик) она подсвечивается голубым
- [ ] Поля ввода имеют темный фон и светлый текст

### ✅ Поля ввода
- [ ] Input поля имеют темный фон (`#121212`)
- [ ] Текст в input полях светлый (`#e0e0e0`)
- [ ] При фокусе на поле появляется голубая рамка (`#42a5f5`)
- [ ] Select (выпадающие списки) также имеют темный фон

### ✅ Scrollbar
- [ ] Scrollbar имеет темный фон
- [ ] Ползунок scrollbar голубой
- [ ] При наведении на ползунок он становится светлее

### ✅ Режим просмотра
- [ ] Войдите в режим просмотра (через auth.html?company=xxx)
- [ ] Все неактивные поля должны иметь приглушенный вид с полупрозрачностью
- [ ] Цвета должны соответствовать темной теме

## Сравнение с диаграммой Ганта

Переключитесь обратно на вид "Диаграмма Ганта" и убедитесь, что цвета таблицы и диаграммы согласованы:
- [ ] Одинаковый фон
- [ ] Одинаковые цвета текста
- [ ] Одинаковые цвета интерактивных элементов

## Возможные проблемы

### Если таблица не темнеет:
1. Проверьте, что кнопка темной темы работает (должна переключаться между Луной и Солнцем)
2. Откройте консоль браузера (F12) и проверьте, что на `<body>` есть атрибут `data-theme="dark"`
3. Обновите страницу (F5)

### Если некоторые элементы остаются светлыми:
1. Проверьте консоль браузера на наличие ошибок CSS
2. Убедитесь, что файл `implementation_schedule.html` сохранен с последними изменениями

## Результат

Все элементы детальной таблицы задач должны правильно отображаться в темной теме, создавая единообразный интерфейс с диаграммой Ганта.
